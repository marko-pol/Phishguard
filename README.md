# PhishGuard

A zero-budget, open-source phishing email classifier powered by traditional ML (scikit-learn + XGBoost). Paste an email and get an instant verdict, confidence score, and plain-English explanation of the red flags detected.

**Live demo**: [PhishGuard on Hugging Face Spaces](https://huggingface.co/spaces/markopolo845/phishguard)

---

## What it does

- Classifies emails as **phishing** or **safe** with a confidence score
- Highlights red flags — suspicious URLs, urgent language, spoofed senders, and more
- Shows a signal breakdown so you can understand *why* the model flagged it

---

## Quick start

**Requirements**: Python 3.10+

```bash
git clone https://github.com/your-username/phishguard.git
cd phishguard

python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate

pip install -r requirements.txt
```

### Train the model

```bash
make train
```

This reads from `data/raw/`, processes the emails, extracts features, and saves the trained model to `models/`.

### Run the app

```bash
make app
```

Then open [http://localhost:7861](http://localhost:7861) in your browser.

### Run tests

```bash
make test
```

---

## Usage

1. Paste the full email (headers + body) into the text box, or click **Load phishing example** / **Load safe example** to try it out
2. Click **Analyze Email**
3. The result card shows:
   - **Verdict** — phishing or safe
   - **Confidence** — how certain the model is (0–100%)
   - **Analysis** — specific red flags found in the email
   - **Signal breakdown** — individual feature scores (URLs, urgency, structure, etc.)

## Datasets

The following datasets need to be downloaded and placed in `data/raw/` before training:

| Dataset | Source |
|---|---|
| SpamAssassin Public Corpus | [spamassassin.apache.org](https://spamassassin.apache.org/old/publiccorpus/) |
| Nazario Phishing Corpus | [github.com/rf-peixoto/phishing_pot](https://github.com/rf-peixoto/phishing_pot) |
| Enron Email Dataset | [kaggle.com/datasets/wcukierski/enron-email-dataset](https://www.kaggle.com/datasets/wcukierski/enron-email-dataset) |
